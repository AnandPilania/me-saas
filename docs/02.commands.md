# Commands

## Init project

```shell
yarn init -y
```

## Init typescript

```shell
npx tsc --init
```

## Init husky

```shell
npx husky-init

# Create commit-msg hook. Pass empty string or any command.
npx husky add .husky/commit-msg ''
```

## Generate token

```shell
# Generate 32 character length token
yarn token
```
